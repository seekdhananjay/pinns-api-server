# pinns-api-server

## How to Setup
Fill you configurations for github and mongo with the following environment details in .env file:

GITHUB_KEY=
GITHUB_SECRET=
SESSION_SECRET=

MONGODB_PIN_USERNAME=
MONGODB_PIN_PASSWORD=
MONGODB_PIN_DATABASE=
MONGODB_HOST=
MONGODB_PORT=

## How to Run
```sh
sh run.sh
```
